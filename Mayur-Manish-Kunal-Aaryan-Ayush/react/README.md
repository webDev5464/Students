- check node version

```bash
node -v
```

- check npm version

```bash
npm -v
```

- install react

Create new folder
```bash
npm create vite 
```

install same folder
```bash
npm create vite ./
```

- install node_modules (dependency's)

```bash
npm install
or 
npm i
```

- start localhost

```bash
npm run dev
```

- Install Router

```bash
npm i react-router-dom
```

- react icons

```bash
npm i react-icons
```
