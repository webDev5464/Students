**Step 1** Open terminal and write :- 

```
npm init -y
```

**Step 2** Add Start script

```json
"scripts": {
    "start": "nodemon script.js",
    "test": "echo \"Error: no test specified\" && exit 1"
},
```

**Step 3** Install nodemon

```
npm i nodemon
```

**Step 4** Run the script

```
npm start
```
